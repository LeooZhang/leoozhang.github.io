蔬菜店会买水果，水果店为什么不卖蔬菜？

发表于 2019-06-30 分类于 认知
本文字数： 0

前言

在36氪看到了一篇“日常水文”，叫“为什么卖菜的店会卖水果，而卖水果的店不卖菜？”，一个产品经理朋友点赞过，拜读之后，有以下想法想分享。

想法

首先就是文章内容偏离标题有点多了..

卖菜会卖水果，卖水果的为什么不卖菜？

这就是货源问题啊，这个卖菜的有水果货源，店里品类越多，受众就越广，水果跟蔬菜的陈列存放条件又类似，当然就也卖点水果啦..

卖水果的为什么不卖菜？超市不都是既有水果也有蔬菜么，而且往往也挨着放…

独立水果店为什么不卖菜？这涉及到种植、供货商等问题，真不是随随便便就能搞定蔬菜货源的…

百果园不卖蔬菜是因为人家业务战略问题啊，也卖菜的话就是“百果菜园”了是不？

不知道为啥文中分析都是从C端这一侧出发的，覆盖的面挺广，甚至让我在读文字的时候脑中想起了Sherlock发动“记忆宫殿”时的BGM…

最后我想说，想知道答案的话，弄个小调研不比自己在这“意淫”强？
